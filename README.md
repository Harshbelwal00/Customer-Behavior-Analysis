# Customer-Behavior-Analysis
# 📊 Data Analytics Project

## 🔹 Overview

This project demonstrates an end-to-end data analytics workflow — from raw data to actionable insights. It includes data loading, cleaning, exploratory data analysis (EDA), SQL querying, dashboard creation, and final reporting.

The objective is to extract meaningful insights and present them in a clear, visual, and business-friendly way.

---

## 🔹 Dataset

* The dataset is loaded and processed using Python in Google Colab
* Contains structured data with numerical and categorical features
* Data issues like missing values, duplicates, and inconsistencies were handled during preprocessing

---

## 🔹 Tools & Technologies

* **Google Colab (Python)** — Data loading, cleaning, and EDA
* **Pandas, NumPy** — Data manipulation
* **Matplotlib, Seaborn** — Data visualization
* **MySQL** — SQL queries and analysis
* **Power BI** — Interactive dashboard
* **Gamma** — Presentation (PPT creation)

---

## 🔹 Steps Performed

### 1. Data Loading (Google Colab)

* Imported dataset using Pandas
* Checked data structure, types, and summary statistics

### 2. Data Cleaning

* Handled missing values
* Removed duplicates
* Fixed data types
* Standardized formats

### 3. Exploratory Data Analysis (EDA)

* Performed statistical analysis
* Created visualizations (bar charts, line charts, heatmaps)
* Identified trends and patterns

### 4. SQL Analysis (MySQL)

* Uploaded cleaned data into MySQL
* Performed:

  * Aggregations
  * Filtering
  * Joins
  * Grouping

### 5. Dashboard Creation

* Built an interactive Power BI dashboard
* Included KPIs, filters, and dynamic visuals

### 6. Reporting & Presentation

* Created a structured report
* Designed a presentation using Gamma
* Focused on insights and recommendations
